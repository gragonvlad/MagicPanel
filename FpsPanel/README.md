# Fps Panel

## About
Displays the current server fps

 ```json
  "Update Rate (Seconds)": 5.0,
  "Panel Settings": {
    "Dock": "leftmiddle",

    "Order": 3,
    "BackgroundColor": "#fff2df08"
  },
  "Panel Layout": {
    "Image": null,
    "Text": {
      "Text": "(S)FPS: {0}",
      "FontSize": 14,
      "TextAnchor": "MiddleCenter",
      "Color": "#33B5E6FF",
      "Order": 1,
      "Width": 1.0,
      "Padding": {
        "Left": 0.05,
        "Right": 0.05,
        "Top": 0.05,
        "Bottom": 0.05

  }
}
 ```
