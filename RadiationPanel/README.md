# Heli Panel

## About
Change the color of the Radiation Icon when a player is in a radiation zone

## Configuration
 
 ```json
{
  "Active Color": "#FFFF00FF",
  "Inactive Color": "#FFFFFF1A",
  "Panel Settings": {
    "Dock": "right",
    "Width": 0.02,
    "Order": 7,
    "BackgroundColor": "#FFF2DF08"
  },
  "Panel Layout": {
    "Image": {
      "Url": "http://i.imgur.com/owVdFsK.png",
      "Enabled": true,
      "Color": "#FFFFFFFF",
      "Order": 0,
      "Width": 1.0,
      "Padding": {
        "Left": 0.05,
        "Right": 0.05,
        "Top": 0.05,
        "Bottom": 0.05
      }
    }
  }
}
 ```
