# Crash Panel

## About
Change the color of the Plane Crash icon when a Plane Crash is active. 

## Configuration
 
 ```json
{
  "Active Color": "#00FF00FF",
  "Inactive Color": "#FFFFFF1A",
  "Update Rate (Seconds)": 5.0,
  "Panel Settings": {
    "Dock": "right",
    "Width": 0.02,
    "Order": 1,
    "BackgroundColor": "#FFF2DF08"
  },
  "Panel Layout": {
    "Image": {
      "Url": "https://i.imgur.com/ipLC2ww.png",
      "Color": "#FFFFFFFF",
      "Order": 0,
      "Width": 1.0,
      "Padding": {
        "Left": 0.1,
        "Right": 0.1,
        "Top": 0.05,
        "Bottom": 0.05
      }
    },
    "Text": null
  }
}
 ```
