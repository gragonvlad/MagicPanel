# Halloween Panel

## About
Change the color of the Halloween icon when the halloween event is active

## Configuration
 
 ```json
{
  "Active Color": "#EB6123FF",
  "Inactive Color": "#FFFFFF1A",
  "Panel Settings": {
    "Dock": "center",
    "Width": 0.02,
    "Order": 13,
    "BackgroundColor": "#FFF2DF08"
  },
  "Panel Layout": {
    "Image": {
      "Url": "https://i.imgur.com/3fTHEiz.png",
      "Enabled": true,
      "Color": "#FFFFFFFF",
      "Order": 0,
      "Width": 1.0,
      "Padding": {
        "Left": 0.05,
        "Right": 0.05,
        "Top": 0.05,
        "Bottom": 0.05
      }
    }
  }
}
 ```
