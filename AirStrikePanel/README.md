# Air Strike Panel

## About
Change the color of the Air Strike icon when an Air Strike is active.

## Configuration
 
 ```json
{
  "Active Color": "#00FF00FF",
  "Inactive Color": "#FFFFFF1A",
  "Update Rate (Seconds)": 5.0,
  "Panel Settings": {
    "Dock": "right",
    "Width": 0.02,
    "Order": 1,
    "BackgroundColor": "#FFF2DF08"
  },
  "Panel Layout": {
    "Image": {
      "Url": "https://i.imgur.com/h3ayf2x.png",
      "Enabled": true,
      "Color": "#FFFFFFFF",
      "Order": 2,
      "Width": 1.0,
      "Padding": {
        "Left": 0.1,
        "Right": 0.1,
        "Top": 0.05,
        "Bottom": 0.05
      }
    }
  }
}
 ```
